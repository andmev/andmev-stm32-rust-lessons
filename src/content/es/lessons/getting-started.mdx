# Introducción

Bienvenido al curso de **STM32 Rust**. En esta lección, cubriremos los conceptos básicos de la programación embebida con Rust en microcontroladores STM32.

Este es un párrafo con algo de _énfasis_ y **texto en negrita**. También puedes usar ~~tachado~~ para texto eliminado.

## Requisitos previos

Antes de comenzar, asegúrate de tener lo siguiente:

1. Una computadora (macOS, Linux o Windows)
2. Una placa de desarrollo STM32
3. Un cable USB
4. Conocimientos básicos de programación en Rust

### Requisitos adicionales

- Cadena de herramientas de Rust instalada
- Gestor de paquetes `cargo`
- STM32CubeIDE o entorno de desarrollo similar

## Ejemplos de código

Aquí hay un ejemplo de código en línea: `println!("¡Hola, STM32!")`.

Y aquí hay un bloque de código:

```rust
fn main() {
    let mut led = Led::new();
    led.toggle();
    println!("¡LED alternado!");
}
```
